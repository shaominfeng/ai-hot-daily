# AI HOT 每日晨报归档

每日自动抓取的 [AI HOT](https://aihot.virxact.com/daily/2026-07-14) 日报，渲染为单文件 HTML 仪表盘，按月份分文件夹归档。

🌐 **在线阅读（GitHub Pages）**：https://shaominfeng.github.io/ai-hot-daily/

## 月度索引
- [2026-07](2026-07/) — 7 月（[在线版](https://shaominfeng.github.io/ai-hot-daily/2026-07/)）

## 目录结构
```
ai-hot-daily/
├── README.md
├── index.html
├── 2026-07/
│   ├── README.md
│   ├── index.html
│   └── ai-hot-daily-YYYY-MM-DD.html
└── ...
```

## 怎么读
- **本地**：双击任意 `ai-hot-daily-YYYY-MM-DD.html`。
- **在线**：用上方 GitHub Pages 链接，HTML 完整渲染。
