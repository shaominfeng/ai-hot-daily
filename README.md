# AI HOT 每日晨报归档

每日自动抓取的 [AI HOT](https://aihot.virxact.com) 日报，渲染为单文件 HTML 仪表盘，按月份分文件夹归档。

🌐 **在线阅读（GitHub Pages，HTML 像网站一样渲染）**：https://shaominfeng.github.io/ai-hot-daily/

## 月度索引
- [2026-07](2026-07/) — 7 月（[在线版](https://shaominfeng.github.io/ai-hot-daily/2026-07/)）

## 目录结构
```
ai-hot-daily/
├── README.md              # 本文件（仓库总索引）
├── 2026-07/
│   ├── README.md          # 本月索引（GitHub 原生渲染）
│   ├── index.html         # 本月索引（在线版着陆页）
│   ├── ai-hot-daily-2026-07-08.html
│   └── ...
└── 2026-08/
    └── ...
```

## 怎么读
- **本地**：直接双击任意 `ai-hot-daily-YYYY-MM-DD.html`，无需联网。
- **在线**：点上面的 GitHub Pages 链接，HTML 会完整渲染（卡片网格、锚点导航都在）。
- 仓库里的 `*.html` 在 GitHub 文件视图中显示的是源码；要看渲染效果请用 GitHub Pages 链接，或用 https://htmlpreview.github.io/ 预览。
